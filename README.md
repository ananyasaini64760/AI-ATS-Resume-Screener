# AI ATS Resume Screening System

This project is an AI-powered ATS Resume Analyzer that evaluates how well a resume matches a job description.

## Features
- PDF Resume Upload
- ATS Match Score (TF-IDF + Cosine Similarity)
- Missing Skills Detection
- AI-based Resume Improvement Suggestions
- Downloadable PDF Report
- Streamlit Web Interface

## Technologies Used
- Python
- NLP (TF-IDF, Cosine Similarity)
- OpenAI API
- Streamlit
- ReportLab

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run app:
streamlit run app.py
